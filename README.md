# Gamers Club
## Demo
This app is deployed on <https://gamers-club.herokuapp.com>

## Introduction
A web application that let gamers share their interesting experience when playing games.
## Features
- Browse all posts
- Browse posts by category
- User register
- User login
- Create a post
- View a post
- Change a post
## Technologies
- HTML, CSS, JavaScript
- Node.js
- MySQL
- BootStrap
- Express
- EJS
- Passport
